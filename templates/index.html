About Us

We are Group 2 and for our final project, we decided to make a Waste Classification API.

The waste classification API utilizes a machine learning model to classify different types of waste (recyclable, biodegradable, nonbiodegradable, etc.) using images. It aims to automate the waste sorting process and help waste management systems segregate and dispose of different kinds of garbage properly. Our API can also provide a list of drop-off locations near the user for recycling. To get started, you only need to make an account. Registered users have access to the following functions:

  • Classify
  • Drop-off Locations
  • History
  • Portfolio

This API was last updated on 07/28/2023.

Frequently Asked Questions

  • What are the categories available for waste classification?

    We classify many different types of waste. These waste types include: electronics, glass, medical, metal, organic waste, paper, cardboard, plastic, textiles, and wood.

  • Within what city are the drop-off locations?

    All drop-off locations used in our API were pulled from a dataset based on New York City zones.

  • How can I see all the previously classified waste images?

    After logging in, you can view the details of your previously classified images under the "History" page. This can be filtered with a text input and by ID, waste type, category, or date. If you could like to see a visualization of your classification history, you can find this under the "Portfolio" page.

  • Can I access the waste classification and management tools without an account?

    No, you must make an account to access all the functions of the API.

  • Is there a cost for using the tools?

    Nope! All you need is an account and you can use any tool on the API.

  • What type of model was built for the waste classification?

    The waste classification model is a tensorflow object detection neural network trained to recognize specific categories of waste.
