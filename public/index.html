About Us

We are Group 2 and for our final project, we decided to make a Waste Classification API.

The waste classification API will utilize machine learning models to classify different types of waste (recyclable, biodegradable, nonbiodegradable, etc.) using images. It aims to automate the waste sorting process and help waste management systems segregate and dispose of different kinds of garbage properly. Our API can also provide a list of drop-off locations near the user for recycling. To get started, you only need to make an account. Registered users have access to the following functions:

  • Classify
  • Drop-off Locations
  • History
  • Portfolio

This API was last updated on 07/13/2023.

Services

Our API provides various services that makes waste management accessible for the everyday user.

Waste Classification

Simply upload images of waste, and our machine learning algorithms will accurately classify them into categories such as recyclable, biodegradable, non-biodegradable, and hazardous

Drop-off Locations

Find nearby drop-off locations for recycling different types of waste, helping you responsibly dispose of your recyclable materials.

Learn about Waste

Access detailed information about various waste items, including descriptions, recycling instructions, and proper disposal methods.

User Profiles

Create an account to track your waste classification history, contribute to waste management initiatives, and stay informed about environmental news.
